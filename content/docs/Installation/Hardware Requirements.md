# Hardware/Software Required
---

![Raspberry Pi](/pictures/Raspberry-Pi.png)
## Raspberry PI
- Raspberry Pi Zero or Zero 2 W (Zero 2 W recommended for stability)
    - Only one network interface, needs usb to ethernet (recommended) or usb to  wifi adapter
    - $25
- Raspberry Pi 3B or newer
    - These models have a built-in Wifi and and Ethernet port so no additional adapters needed
    - $46 to $128

{{< hint info >}}
If you have a choice, start with a Pi 3B or newer for simpler wiring
{{< /hint >}}
