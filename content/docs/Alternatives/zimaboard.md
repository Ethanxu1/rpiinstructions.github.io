# ZimmaBoard
---
- A [ZimaBoard](https://www.zimaspace.com/products/single-board-server) is a fanless, x86-based SBC designed for DIYers and developers to build custom servers, media centers, and other projects.
- **Price**
    - $120 - $200
- **Pros**: 
    - Often come with multiple Ethernet ports.
    - Can run full router distributions (pfSense, OPNsense, OpenWrt, etc.).
    - Very flexible and powerful.
- **Cons**: More expensive and power-hungry than a Raspberry Pi.
    - More expensive and more power-hungry than a Pi.
    - Overkill if all you need is a simple home or travel router.
